Expert System Architecture

All Core Components Implemented

LogiCore is a complete rule-based expert system with every standard component fully operational.

Knowledge Base

Stores 20+ expert rules and domain facts encoding supervisor knowledge about project selection, skill requirements, and complexity levels.

Inference Engine

Applies forward chaining to iteratively fire rules until a fixpoint is reached, deriving new facts from the student's profile inputs.

Forward Chaining

Data-driven reasoning that starts from known facts (GPA, skills, time) and fires applicable rules to build conclusions bottom-up.

Explanation Facility

Each recommendation is fully justified. The system shows every rule that fired, why it applied, and how it influenced the final result.

Student Interface

A multi-step guided consultation form captures student skills, GPA, career goals, time availability, and priority preferences.

Scored Rankings

Projects are scored across multiple dimensions — skills, career fit, complexity, risk, novelty — and presented in ranked order with match percentages.
